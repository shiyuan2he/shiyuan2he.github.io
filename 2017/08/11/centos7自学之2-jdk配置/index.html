centos7自学之2-jdk配置

本章讲解在centos服务器上部署java开发环境之jdk安装，分rpm安装和tar.gz安装

安装之前

安装环境检查

查看当前系统jdk

java -version

进一步查看jdk信息

rpm -qa | grep java

rpm -qa | grep jdk

rpm -qa | grep -E ‘^open[jre|jdk]|j[re|dk]

卸载jdk

rpm -e –nodeps + 上面查出来的openjdk相关

rpm安装

下载rpm安装文件

进oracle官网下载适用于自己版本的*.rpm格式的jdk
wget方式下载，进入保存下载文件的路径

cd /home/robot/data

wget http://download.oracle.com/otn-pub/java/jdk/8u144-b01/090f390dda5b47b9b721c7dfaa008135/jdk-8u144-linux-x64.rpm

如果没有wget命令，执行

yun install wget

rpm命令安装

rpm -ivh *.rpm

验证安装，终端输入 java 或者 javac 或者 java -version

java

或者

javac

或者

java -version

出现下图情况即为安装成功



注意：此种安装方式不需要配置环境变量，比较快捷，方便！此方式也是我目前使用的方式

tar.gz解压安装

解压jdk

tar -xzvf jdk.tar.gz
mv jdk /usr/local/

编辑/etc/profile

使更改生效

source /etc/profile
